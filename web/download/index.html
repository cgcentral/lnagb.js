The lnagb.js library

The lnagb.js JavaScript library is ready to be used in your applications. The first link points to the uncompressed format, while the second points to the compressed (minified) format. Left click to download, right click to copy.

https://novakcgx.me/lnagb.js/lib/lnagb.js
https://novakcgx.me/lnagb.js/lib/lnagb.min.js

Being ‚Äúuncompressed‚Äù means the code is readable and can be inspected, and is best for the development of the library. The compressed format provides better performance, on the other hand, and is best for use in production or with real applications.
